Publishing tool

MEL publishing tool. This page gives access to MEL module developers to start a publish process. In doing so, the MEL website will be automatically updated with new changes made since the last publish.

To start the process, complete the following:

  1. Make sure you have a good internet connection
  2. Click the button below, a new tab will appear.
  3. Follow on-screen directions to authenticate yourself to Google
  4. Watch as the publisher completes it's process
  5. Confirm the new version of the website reflects your most recent changes.
