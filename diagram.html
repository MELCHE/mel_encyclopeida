Diagram Creator

This tool is used to create interactive, hyperlinked diagrams for use on Application pages in the encyclopedia.

Start over

First, upload an image of the diagram.

Next, create hyperlink regions on the diagram. Undo Done

Finally, add a URL for each link region. Done

Make sure you enter valid URLs for every region.

    To Create a Section "menu" Page:

    1. Download this code file.

    2. Upload the file as-is to the correct location in Google Drive.

    To embed into a module:

    1. Open the Google Document of the module

    2. Place the insertion cursor (|) in the place you want to insert the diagram

    3. From the toolbar, select "Insert" > "Link".

    4. In the text box that appears, enter the following "code" snippet for the "Text" field:

      {{DIAGRAM: }}

    5. In the "Link" field enter the following URL:

      http://encyclopedia.che.engin.umich.edu

      and then press enter
